The Da Nang map screenshot used in the "Discover" section of the landing page.

**To replace:**
1. Take a Google Maps screenshot showing partner clubs across Đà Nẵng (zoom level so 10–15 pins are visible)
2. Crop to roughly 4:3 aspect ratio
3. Export as JPEG, quality ~85%, target width 1200–1600 px
4. Save as `clubs-danang.jpg` in this folder, overwriting any placeholder

**Recommended tools:**

- Or: Screenshot → ImageOptim / TinyJPG for compression

Target file size: under 200 KB.
